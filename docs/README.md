# كتيّبات مُراح — دليل الاستخدام الكامل

ثلاثة كتيّبات تشرح تطبيق «مراح» لإدارة الحلال من البداية حتى النهاية، كلٌّ حسب نوع المستخدم.
كل بند في الكتيّبات مكتوب بالصيغة: **العنوان كما في الموقع · الفكرة · المختصر · التفصيل**.

## الكتيّبات

1. **[👤 دليل الزائر](دليل-الزائر.md)**
   كل ما يخص حساب الزائر: التسجيل والدخول، الرئيسية، النصائح والمعلومات، المنتدى كاملاً، الدعوة لمشاهدة حلالٍ (عرض فقط)، والخصوصية.

2. **[🐑 دليل أصحاب الحلال](دليل-أصحاب-الحلال.md)**
   إدارة الحلال حسب الصلاحيات: الحلال، إضافة/تعديل بهيمة، سجل البهيمة، التلقيح والولادات، التطعيمات، العلاجات، عمليات بالجملة، التنبيهات، النسخ الاحتياطي، ومشاركة الحلال.

3. **[🛡️ دليل الإدارة](دليل-الإدارة.md)**
   أدوار الإدارة الثلاثة (مدير المراح، مدير النظام، مشرف المنتدى): المستخدمون والصلاحيات، أنواع الحلال، سلة المحذوفات، النصائح والمعلومات، إعدادات المنتدى، الاستعادة الكاملة، ونموذج الخصوصية والأمان.

## ملاحظات عامة

- **أنواع الحسابات:** «🐑 صاحب حلال» (يحتاج موافقة المدير) و«👤 زائر» (دخول فوري). والإدارة أدوار إضافية تُمنح من المدير.
- **مبدأ الصلاحيات:** ما يراه المستخدم يعتمد على صلاحياته (عرض/إضافة/تعديل/حذف لكل قسم).
- **الخصوصية:** كل صاحب حلال يرى حلاله فقط، وبيانات الأعضاء سرية، والمشاركة بدعوةٍ وقبول متبادل (عرض فقط).
- **الدخول:** بالجوال أو اسم المستخدم + رقم سري من ٤ أرقام. الخروج بزر **⎋** أعلى الشاشة.
